Guía de Indicaciones para el Código del Producto

Para que el código del producto sea válido, debe cumplir con los siguientes requisitos:

  • Mínimo 8 caracteres de longitud.
  • Debe contener al menos una letra minúscula y una letra mayúscula.
  • Debe contener al menos dos números.

Ejemplo de Códigos Válidos

Ejemplo 1: Abc12345

Ejemplo 2: XyZ789aa

Ejemplo 3: Test1234

Ejemplo 3: Test1234h

Por favor, asegúrate de ingresar un código que cumpla con estos requisitos para registrar correctamente el producto.
